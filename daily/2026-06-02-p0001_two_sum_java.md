## Problem

- Number: 1
- Title: Two Sum
- Link: https://leetcode.com/problems/two-sum/
- Difficulty: Easy
- Language: Java
- Attempt type: guided practice

## Initial Approach

For each number, compute the missing value needed to reach `target`.
That missing value is `target - number`.

## Final Approach

Use a `HashMap<Integer, Integer>` to store `number -> index`.

1. Loop through `nums` with index `i`.
2. Read the current `number`.
3. Compute `complement = target - number`.
4. If `complement` already exists in the map, return its stored index and `i`.
5. Otherwise store the current `number` and index.

## Complexity

- Time: `O(n)`
- Space: `O(n)`

## Mistakes and Review

- `complement` means `target - number`, not total sum minus number.
- Java map syntax: `Map<Integer, Integer> numMap = new HashMap<>();`
- `put(key, value)` stores data.
- `get(key)` retrieves data.
- `containsKey(key)` checks whether a key exists.
