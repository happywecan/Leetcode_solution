# SQL Solutions

Store reviewed SQL solutions by difficulty:

```text
easy/
medium/
hard/
```

Naming convention:

```text
p<number>_<problem_slug>.sql
```

Example:

```text
easy/p0175_combine_two_tables.sql
```

Each file should include:

- the LeetCode problem link
- the final SQL query
- the main concept used
- a short explanation when the choice between `INNER JOIN`, `LEFT JOIN`,
  aggregation, CTE, or a window function matters
