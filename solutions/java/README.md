# Java Solutions

Store reviewed Java solutions by difficulty:

```text
easy/
medium/
hard/
```

Naming convention:

```text
P<number>_<ProblemName>.java
```

Example:

```text
easy/P0001_TwoSum.java
```

Each file should include:

- the LeetCode problem link
- the final implementation
- time complexity
- space complexity
- a short note when the solution was hint-assisted or reviewed with assistance
